pycomma v0.1.0
authored by sean hwang
pycomma
A python library designed to allow csv data processing workflows completely within a python shell
Installation
Requires python version newer than or equal to 3.6
pip install pycomma
or if using within shell,
pip install --user pycomma
Quickstart Example

Linux/ios

$python3

Windows

> python

>>> from pycomma.comma import Comma

>>> comma = Comma("data.csv", includes_header=True)

>>> comma.prepare()

Preparation complete

>>> comma.set_config("max_row_display", 10)

>>> comma.assign_primary("id")

>>> comma.sort_by_column("id")

>>> comma.show()

Configurations
Following can be changed via set_config:
success_messages : True or False : turns on/off success messages after successful function calls
max_row_display : integer : set number of rows to be shown when show() is called

change
Description: changes matching values in a column to a new value
change(column_name, changing, change_to, case_matters=False)

column_name : string : name of column to look within for value to change

changing : string : value being changed

change_to: string : new value that will replace values matching changing

case_matters : boolean : set True to make case sensitive when searching for value matches
